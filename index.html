Face Recognition

Projects related to Face Recognition

This project is made with the vision of expanding it to cover face recognition as whole and not focus on any single algorithm.
The following are the features of the project:

  • Uses highly optimized Scikit Learn's PCA class.
  • Demonstration of how eigenfaces can be used for face recognition.
  • Achieved accuracy of 88.8%
  • An interactive streamlit app (Face Recognition Lab) to explore the project.
